## 🚀 Featured Project: EmotiNest

**AI-powered HealthTech platform designed to help families with neurodivergent children manage daily emotional needs.**

EmotiNest is a real-world product where I built and applied AI in a sensitive, high-trust environment.

### What I built:
- Designed the full user experience and daily interaction model  
- Built an AI chatbot (EmotiCoach) using LLMs for personalized guidance  
- Developed backend logic and data handling using Supabase  
- Took the product from idea to a live mobile app (iOS & Android)  

👉 This project reflects my approach: building AI systems that are not just functional, but usable and reliable.
